# luna
以Spring为核心的分布式开发框架，致力于解决企业的一站式开发问题。

技术栈：
1.Spring      核心框架
2.SpringMvc   控制层框架
3.MyBatis     持久层框架
4.redis       缓存框架
5.quartz      定时调度框架
6.httpclient  分布式服务Http协议框架
7.Dubbo       阿里巴巴集团分布式服务治理框架
8.zookpper    注册中心
9.solr        搜索中间件
10.ActivieMQ  消息中间件
11.MySql      数据库
12.sso        单点登录
